C

Para que serve: Sistemas operacionais, embarcados.

Exemplo: Linux, drivers.

C++

Para que serve: Jogos, simuladores, tempo real.

Exemplo: Unreal Engine.

Java

Para que serve: Aplicativos Android, empresas.

Exemplo: Android, sistemas bancários.

Python

Para que serve: IA, web, automação.

Exemplo: TensorFlow, Flask.

JavaScript

Para que serve: Web dinâmica, front/back-end.

Exemplo: React, Node.js.

Ruby

Para que serve: Web com Rails.

Exemplo: GitHub, Airbnb.

Swift

Para que serve: Apps iOS/macOS.

Exemplo: Instagram (iOS).

PHP

Para que serve: Web backend.

Exemplo: WordPress.

Go (Golang)

Para que serve: Serviços web rápidos.

Exemplo: Google Cloud.

R

Para que serve: Estatísticas e dados.

Exemplo: Finanças, ciência.

SQL

Para que serve: Banco de dados.

Exemplo: MySQL, PostgreSQL.

HTML

Para que serve: Estrutura de páginas web.

Exemplo: Sites, blogs.

CSS

Para que serve: Estilização web.

Exemplo: Layouts, temas.

MATLAB

Para que serve: Matemática e engenharia.

Exemplo: Simulações científicas.
